# triggertroll

GitHub Pages hosted spam redirect page for FiveM name injection testing.

## Usage

Inject this into FiveM player name:
```html
<meta http-equiv='refresh' content='1; url=https://bader133b.github.io/triggertroll/'>
```

## GitHub Pages URL

Once GitHub Pages is enabled, the page will be available at:
`https://bader133b.github.io/triggertroll/`
